아래는 기존 지침에 추가로, 더 풍부하고 세부적인 분석을 이끌어낼 수 있는 개선된 프롬프트 예시입니다. 각 JSON 필드마다 최소 4문장에서 최대 9문장가량의 길고 자세한 한국어 존댓말 문장으로 답변을 작성해 달라는 요구사항을 명확히 하고 있습니다. 이 프롬프트를 사용하면 생성되는 결과물이 보다 장문의 내용과 세분화된 분석을 담을 수 있습니다.

## [개선된 프롬프트 예시]

다음 지침에 따라 낙상 위험 분석을 위해 이미지를 평가해 주세요. 분석 시에는 **이미지에 실제로 보이는 요소**만 언급하며, 어떠한 추측도 하지 말아주세요. **반드시 각각의 JSON 필드에 대해서 4~9문장 정도의 길고 자세한 한국어 존댓말 문장**으로 작성해 주시기 바랍니다.

### 1. 방 정보
- **Room Name**: `${RoomName}`
- **Room Category**: `${RoomCategory}`

### 2. 평가 기준
1. **Image Description (imageDescription)**
   - 이미지를 전체적으로 묘사해 주십시오.
   - 방 안에 있는 가구나 구조물, 천장 높이, 조명 상태 등 주요 요소를 상세하게 기술해 주십시오.
   - 창문이 있다면 크기와 위치, 두꺼운 커튼이나 블라인드가 있는지 여부를 자세히 적어주세요.
   - 추가로 침대의 위치나 크기, 조명 스위치의 위치, 주변 물건들의 배치 상황까지 구체적으로 서술해 주십시오.

2. **Obstacle Evaluation (fallAnalysis.obstacles)**
   - 바닥 위에 존재하는 장애물을 구체적으로 파악해 주세요. 예: 전선, 카펫, 가방, 옷가지 등.
   - 장애물이 몇 개나 있고, 어느 위치에 놓여 있으며, 크기가 어느 정도인지 자세히 서술해 주십시오.
   - 가구 사이의 간격이나 문 앞, 복도에 쌓여 있는 물건 등 이동 경로상 장애물을 조사해 주십시오.

3. **Floor Condition Evaluation (fallAnalysis.floorCondition)**
   - 바닥이 젖어 있거나 미끄러운지, 혹은 균열이 있는지 상세히 기술해 주세요.
   - 카펫이 있으면 그 두께와 미끄럼 방지 여부, 바닥재의 재질 등도 구체적으로 작성해 주십시오.
   - 햇빛의 반사 정도나 조명의 빛 번짐 등으로 인해 눈부심이나 바닥이 잘 보이지 않는 부분은 없는지 기술해 주세요.

4. **Other Factors (fallAnalysis.otherFactors)**
   - 침대 또는 의자가 스위치나 창문, 화장실과 얼마나 가까이 위치하는지 적어 주세요.
   - 눈에 띄는 문턱 혹은 단계 차이가 있는지, 특히 화장실 문턱이 높아 이동 시 주의가 필요한지 살펴봐 주세요.
   - 화장실 내부가 보인다면, 안전손잡이(Grab Bars)나 미끄럼 방지 매트 유무 등도 언급해 주십시오.
   - 전반적으로 안전을 저해하는 추가 요소(예: 조도 부족, 무질서한 전선 배치, 제대로 고정되지 않은 가구 등)가 있다면 함께 서술해 주십시오.

### 3. 낙상 예방 우선순위
- **화장실 바닥 상태**가 첫 번째 우선순위입니다.
- 그 다음으로 **화장실 입구 문턱**이 중요한 체크 포인트입니다.
- 집 안팎의 **계단**도 반드시 살펴봐 주세요.
- 욕조가 보인다면, **욕조의 존재**만으로도 낙상 위험이 추가된다고 봅니다.

### 4. 최종 출력 형식
아래 JSON 형식을 **엄격하게 지켜** 작성해 주세요. 
- 모든 값(value) 필드는 반드시 한국어 존댓말(높임말)로 작성해 주십시오.
- 각 필드마다 4~9문장으로 구성된 풍부하고 상세한 문장을 적어 주세요.
- 문자열 이외의 다른 데이터 타입 사용은 지양해 주세요(숫자, 불리언 대신 문자열로 표기).

```json
{
  "imageDescription": "<방을 묘사하는 4~9문장 분량의 설명>",
  "fallAnalysis": {
    "obstacles": "<바닥 장애물 여부와 구체적 특징을 4~9문장 분량으로 작성>",
    "floorCondition": "<바닥 상태 평가를 4~9문장 분량으로 작성>",
    "otherFactors": "<기타 낙상 위험 요소들을 4~9문장 분량으로 작성>"
  },
  "fallSummaryDescription": "<종합 평가 및 요약을 4~9문장으로 작성>",
  "fallPreventionMeasures": [
    "<낙상 예방을 위한 제안 1: 4~9문장>",
    "<낙상 예방을 위한 제안 2: 4~9문장>",
    "<낙상 예방을 위한 제안 3: 4~9문장>"
  ]
}
```
